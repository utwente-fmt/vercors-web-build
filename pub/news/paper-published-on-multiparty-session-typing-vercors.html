Paper published on Multiparty Session Typing & VerCors

2023-02-02

We wish to congratulate Jelle Bouma and his supervisor Sung-Shik Jongmans with publishing their paper at TACAS 2023. It is titled "Multiparty Session Typing in Java, Deductively", and it presents the BGJ tool, which checks adherence of Java code to a multiparty session type with near-zero overhead. It accomplishes this by generating contracts that encode this protocol, which can then be verified with VerCors. A preprint can be found on Sung-Shik's website: https://sungshik.github.io/.
